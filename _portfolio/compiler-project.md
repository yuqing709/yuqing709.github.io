## Compiler Construction Project

A compiler developed as a course project for Compiler Principles, implemented in C.

This project supports the full compilation pipeline from source code to target machine code.

### Features

- Lexical analysis (tokenization)
- Syntax analysis (parsing)
- Intermediate representation generation
- Target code generation
- Multi-stage compilation process

### Responsibilities

- Collaborated with a teammate to design and implement the compiler
- Participated in grammar design and parsing logic implementation
- Implemented parts of the compilation pipeline and debugging

### Tech Stack

- C
- Compiler Design
- Data Structures & Algorithms

### Highlights

- Covers the full compiler pipeline from frontend to backend
- Modular design with clear compilation stages
- Strong focus on correctness and structure
