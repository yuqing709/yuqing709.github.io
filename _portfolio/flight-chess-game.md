## Flight Chess Game

A C-based implementation of the classic Flight Chess board game, with enhanced gameplay mechanics and basic AI behavior.

### Features

- Core game logic implementation (movement, rules, turn system)
- Turn-based gameplay with multiple players
- Customized dice probability system to improve game balance
- Unique potion system adding strategic elements to gameplay
- Simple AI for computer-controlled players

### Responsibilities

- Designed and implemented the overall game architecture
- Developed core gameplay logic and rule system
- Designed custom mechanics (dice probability & potion system)
- Implemented basic AI decision-making for computer players

### Tech Stack

- C
- Data Structures
- Game Logic Design

### Highlights

- Improved player experience through probabilistic tuning of dice outcomes
- Introduced additional strategy via potion mechanics
- Implemented simple AI behavior to simulate human-like gameplay
- Focus on modular and readable code structure
